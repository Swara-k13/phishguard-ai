# 🛡️ PhishGuard AI

**AI-powered phishing detection using machine learning and Flask**

---

## 📌 Overview

PhishGuard AI is a machine learning-based cybersecurity web application that detects phishing URLs in real time by analyzing URL features and identifying malicious patterns. It helps users stay protected from online threats through fast and accurate predictions.

---

## 🚀 Features

* 🔍 Detects phishing URLs in real time
* 🤖 Machine Learning-based classification
* 🌐 Simple and interactive web interface
* ⚡ Fast prediction results
* 📊 Trained on URL-based dataset

---

## 🧠 Tech Stack

* **Backend:** Python, Flask
* **Machine Learning:** Scikit-learn
* **Data Processing:** Pandas, NumPy
* **Frontend:** HTML, CSS

## ⚙️ Installation & Setup

1️⃣ Clone the repository

```bash
git clone https://github.com/Swara-k13/phishguard-ai.git
```

2️⃣ Navigate to the project folder

```bash
cd phishguard-ai
```

3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

4️⃣ Run the application

```bash
python app.py
```

5️⃣ Open in browser

```
http://127.0.0.1:5000/
```
